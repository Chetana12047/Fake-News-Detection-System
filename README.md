📰 Fake News Detection System

An AI-powered Fake News Detection System that uses Machine Learning, NLP, and Business Intelligence concepts to classify news articles as Fake or Real.

📌 Project Overview

The rapid growth of digital media and online news platforms has significantly increased the spread of misinformation and fake news. Manually verifying the authenticity of online content is time-consuming and difficult.

This project focuses on developing an intelligent Fake News Detection System using Machine Learning and Natural Language Processing (NLP) techniques. The system analyzes textual patterns and classifies news content as Fake or Real with a prediction confidence score.

The project also integrates Business Intelligence concepts to extract meaningful insights from textual data and improve information reliability.

⚙️ How It Works

1. User enters news content
2. Text preprocessing is applied
3. Tokenization and lemmatization are performed
4. TF-IDF converts text into numerical vectors
5. Machine Learning model analyzes the content
6. The system predicts whether the news is Fake or Real
7. Prediction confidence score is displayed

⸻

🤖 Models Used

The project experiments with multiple Machine Learning models including:

* Logistic Regression
* Multinomial Naive Bayes
* Random Forest

The best-performing model is deployed using Streamlit for real-time prediction.
